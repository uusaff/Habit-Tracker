# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some Oxlint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the Oxlint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and Oxlint's TypeScript related rules in your project.

# 🌴 Tropical Habit Tracker

A minimalist, visually appealing, and highly interactive habit-tracking application designed to help users build consistency and maintain daily routines effortlessly.

Built with React, Tailwind CSS, and Framer Motion, the app combines a clean user experience with smooth animations and a modern glassmorphism-inspired tropical design.

## ✨ Features

### 🎯 Custom Habit Creation

* Create unlimited habits.
* Choose unique icons and vibrant color themes.
* Personalize your tracking experience.

### 📈 Real-Time Progress Tracking

* Dynamic circular progress indicator.
* Instantly visualize daily completion rates.

### 📅 Weekly Consistency Tracker

* Interactive weekly grid layout.
* Monitor your performance throughout the week.

### 🔥 Streak System

* Track consecutive completion streaks.
* Stay motivated with visual streak indicators.

### 💡 Inspirational Dashboard

* Rotating motivational quotes.
* Encourages positive habit-building and consistency.

### 💾 Persistent Storage

* Data is stored locally using Local Storage.
* Progress remains intact after page refreshes.

### 🎨 Modern Glassmorphism UI

* Tropical-inspired design.
* Responsive layout optimized for both desktop and mobile devices.

## 🛠 Tech Stack

* **React.js**
* **Tailwind CSS**
* **Framer Motion**
* **Lucide React**

## 🚀 Getting Started

```bash
git clone https://github.com/uusaff/habit-tracker.git
cd habit-tracker
npm install
npm run dev
```

## 🎯 Design Philosophy

Tropical Habit Tracker is more than just a habit-recording tool. Its goal is to make habit formation a calming, enjoyable, and motivating experience through thoughtful design, smooth interactions, and meaningful progress visualization.
